## Description

DroneFarm is a programming game where you write simple Python-like commands to control a farming drone. No experience needed—just start coding and watch your drone automate the farm.

**Learn by doing**
- Write real code: `move_right()`, `water()`, `for i in range(5):`
- See your commands execute in real-time on a beautiful farm grid
- Master loops, conditionals, and problem-solving

**13 missions**
- 10 tutorial missions that teach the basics
- 3 advanced optimization challenges
- Earn stars and badges as you progress

**Built for learners**
- Syntax highlighting and hints
- Sample solutions when you're stuck
- Run, step, or reset—learn at your own pace

Perfect for beginners curious about programming, or anyone who wants a fun way to practice logic and automation. Code. Fly. Grow.
